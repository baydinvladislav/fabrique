API позволяет получить информацию об объектах моделей приложения в формате JSON. 

Доступные запросы:

http://127.0.0.1:8000/api/polls/ - получение списка доступных вопросов.
http://127.0.0.1:8000/api/polls/{id}/ - получение конретного опроса по его id.
http://127.0.0.1:8000/api/polls/?search={expiration_date} - фильтрация опросов по дате окончания.


http://127.0.0.1:8000/api/questions/ - получение списка доступных вопросов.
По 5 штук на каждой странице.
http://127.0.0.1:8000/api/questions/{id}/ - получение конкретного вопроса по его id.
http://127.0.0.1:8000/api/questions/?search={answer_type} - фильтрация по вопросов по типу его ответа.
